# Security Policy

## Supported Versions

Security fixes are provided for the latest released version.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

To report a vulnerability responsibly, use the IT security contact information at:

https://invokable.gmbh/en/it-security

Please include:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- A suggested remediation (if available)

We will acknowledge the report and coordinate a fix and disclosure timeline.
